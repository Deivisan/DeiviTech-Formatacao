# Cenário de Sistemas Operacionais — 2025/2026

## O Fim do Windows 10

### O que aconteceu

A Microsoft encerrou oficialmente o suporte ao Windows 10 em **14 de outubro de 2025**:

- ❌ Zero atualizações de segurança
- ❌ Zero correções de bugs
- ❌ Zero suporte técnico da Microsoft
- ⚠️ Vulnerável a malwares e ataques

### Quem é afetado

- Em outubro de 2025, o Windows 10 ainda tinha **~60% de market share** em desktops
- Isso significa que centenas de milhões de PCs ficaram sem proteção
- No Brasil, a situação é ainda mais crítica — muitos PCs são antigos e não suportam Windows 11
- Em Feira de Santana e interior da Bahia, a maioria dos PCs ainda roda Windows 10

### ESU (Extended Security Updates)

- A Microsoft oferece o programa ESU por US$ 30/ano para consumidores
- Fornece atualizações de segurança por mais 3 anos (até 2028)
- Mas não é viável para a maioria dos brasileiros

**Benefícios:**

### 2. Chrome OS Flex

**O que é:** Sistema da Google, baseado em Chrome, gratuito.

**Benefícios:**

- Gratuito total
- Extremamente leve (roda em PCs de 2010)
- Seguro por design (sandbox)
- Atualizações automáticas
- Integrado com Google Workspace

**Para quem:**

- Navegação web + Office online
- Estudantes
- PCs muito antigos (2GB RAM, Pentium)

### 3. Linux Ubuntu/Mint

**O que é:** Sistema operacional open source, poderoso e gratuito.

**Benefícios:**

- 100% gratuito (SO + todos os programas)
- Imune a vírus Windows
- LibreOffice incluso (Word, Excel, PowerPoint)
- Terminal poderoso para programação
- Atualizações por 5 anos (LTS)

**Para quem:**

- Programadores e estudantes de TI
- Empresas que querem reduzir custos
- Usuários básicos (internet + office)

### 4. Windows 11

**O que é:** O Windows mais recente com suporte até 2031.

**Requisitos mínimos:**

- CPU Intel 8ª geração+ ou AMD Ryzen 2000+
- 4GB RAM (8GB recomendado)
- 64GB de espaço
- TPM 2.0

**Nota:** Muitos PCs populares no Brasil NÃO atendem aos requisitos do Windows 11 — por isso, as alternativas acima são essenciais.
